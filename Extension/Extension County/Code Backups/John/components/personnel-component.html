Personnel Component

The purpose of this component is to display personnel from the database and display it in a way that is desired by the content manager.

Other Info/Rules:

  • Please note the .highlighted class in each example.
  • The last example - the "Plain List with Sorting" refers to the user's ability to filter content they see in lists. In this way, a content manager could select to show all personnel for a county, and then allow the user to filter based on category if they so desire.
  • I can build some JavaScript/jQuery sorting functionality for the filtering. Is this somethig you want me to go ahead and do, or do you have a better way to do it for your purposes? Either way is fine, just let me know. I picked a select box for simplicity and for mobile compatability.
  • The plain list items for personnel have an inline list that displays the contact info, but on the tiled list, it is removed due to styling and is just a regular list.

Accessibility:

  • The the Foundation panels should be accessible by default since they are simply styled divs. So as long as we are sure to include an alt tag by default, we should be fine.
