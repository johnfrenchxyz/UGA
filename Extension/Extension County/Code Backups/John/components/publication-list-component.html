Publications List Component

The purpose of this component is to display titles, summary, image, etc of a list of publications from our database, probably sorted by keywords, topics, or characteristics.

Other Info/Rules:

  • How do we want to display the publication number? In the plain list example below, the first piblication has the text "publication no" followed by the number, while the second publication only has the number. I know this may seems specific, but I think it is worth a quick discussion.
  • Should the Read More link be on a different line, or should it be on the same line?
  • Should we have a max character limit for the description/summary section so that all of the publications blocks are mostly uniform?

Accessibility:

  • The the Foundation panels should be accessible by default since they are simply styled divs. So as long as we are sure to include an alt tag by default, we should be fine.
