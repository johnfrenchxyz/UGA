CAES Components:

This is the list of all elements that our users can utilize to customize and build their pages in Adobe Experience Manger. The static mockups of these components have been combined with the Java-based AEM framework to allow content managers to drag and drop these components into their site. The dynamic components pull information from various databases, and the static components can be manuipulated and changed by the users as needed.
