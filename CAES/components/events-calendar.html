CAES Events Calendar:

The purpose of this template component is to give the user the ability to have a page that lists all upcoming events. It has the ability to set a featured event, while still showing upcoming events below. We were able to re-utilize the Event List component and give it a pagination functionality (seen at the bottom of the mockup), so that users could click through the list of upcoming events.
