CAES Event List:

The purpose of this component is to display events from our CAES master calendar. Within Adobe Experience Manager, users will have the ability to apply filters to control what type of events are displayed on their pages. The component is designed to look and work well, no matter what size the screen is - it is fully responsive. Note: Only the fist item has schema.org formatting, so use that as a reference for what will go into Adobe Experience Manager.
